# Perguntas Frequentes (FAQ)

## O que é o nation.fun?
O **nation.fun** é uma plataforma para conectar e criar agentes de IA autônomos para automatizar tarefas.

## Preciso saber programar para usar a plataforma?
Não! Você pode criar agentes usando nossos templates pré-configurados sem escrever uma única linha de código.

## Quais canais são suportados?
Atualmente, suportamos integrações com WhatsApp e Telegram, e estamos trabalhando para adicionar mais canais em breve.
