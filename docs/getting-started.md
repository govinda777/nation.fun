## What You'll Build

In this 10-minute guide, you'll create your first AI agent that responds to Twitter mentions.

## Prerequisites

- A Twitter account
- An email address
- A Web3 wallet (MetaMask, Coinbase Wallet, etc.)

## Step 1: Create Your Account

1. Go to [app.nation.fun](https://app.nation.fun)
2. Click "Sign Up"
3. Choose signup method:
- Email
- Google
- GitHub
- Web3 wallet
4. Complete verification

## Step 2: Choose Your Agent Template

1. Click "Create Agent" on dashboard
2. Browse templates:
- **Twitter Bot** - Engage with your community
- **Trading Bot** - Execute trades automatically
- **Support Agent** - Answer customer questions
- **Blank Canvas** - Custom agent

For this guide, select **Twitter Bot**

## Step 3: Configure Your Agent

### Basic Settings
- **Name:** Give your agent a name (e.g., "Community Manager")
- **Description:** What does your agent do?
- **Avatar:** Choose or upload an image

### Personality
- **Tone:** Professional, Friendly, Casual, etc.
- **Personality:** Traits that define how it responds
- **Response Style:** Verbose, Concise, Creative

### Twitter Integration
1. Click "Connect Twitter"
2. Authorize Nation.fun on Twitter
3. Select which mentions to respond to
4. Set response rules (optional)

## Step 4: Set Rules & Constraints

Define agent behavior:

```
IF user mentions (@bot_name)
AND user is following account
THEN respond with helpful reply
AND like tweet
```

Advanced rules allow complex logic.
