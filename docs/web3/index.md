# Bem-vindo à Nação Web3

Nesta seção, você encontrará tudo o que precisa saber sobre como usamos a tecnologia Web3 para potenciar os Agentes da Nation.fun.

Nosso objetivo é tornar essa tecnologia poderosa acessível a todos, mesmo para quem não tem familiaridade com criptomoedas ou blockchain.

Aqui você aprenderá sobre:

- Como suas chaves e ativos digitais são protegidos com a Privy.
- O que é o "NFT Nation Pass" e como ele lhe dá controle sobre seu Agente.
- Como as taxas de transação funcionam de forma transparente.
- Como você tem total autonomia para mover seus ativos para outras carteiras.

Explore os tópicos para entender como garantimos segurança, transparência e propriedade real dos seus ativos digitais.
