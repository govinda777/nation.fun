# Setup para Desenvolvimento Local

Este guia descreve como configurar o ambiente de desenvolvimento para contribuir com o **nation.fun**.

## Pré-requisitos

-   Node.js 18+
-   npm

## Passos para Instalação

1.  **Clone o repositório:**
    ```bash
    git clone https://github.com/govinda777/nation.fun.git
    cd nation.fun
    ```

2.  **Instale as dependências:**
    ```bash
    npm install
    ```

3.  **Configure as variáveis de ambiente:**
    - Copie o arquivo `.env.example` para `.env` e preencha as variáveis necessárias.

4.  **Execute a aplicação em modo de desenvolvimento:**
    ```bash
    npm run dev
    ```

A aplicação estará disponível em `http://localhost:3000`.
