## Social Media Skills

### Twitter Skills
- **Post Tweet** - Publish content to Twitter
- **Reply to Mentions** - Auto-respond to mentions
- **Engage Content** - Like, retweet, follow based on rules
- **Search & Monitor** - Track keywords, hashtags, users
- **Direct Messages** - Send DMs to users

### Telegram Skills
- **Send Messages** - Post to Telegram channels/groups
- **Read Messages** - Monitor and respond to messages

### Discord Skills
- **Send Messages** - Post to Discord channels
- **Moderation** - Manage messages, users
- **Reactions** - React to messages

## How to Use Skills

1. Go to Agent Builder
2. Click "Add Skill"
3. Select skill category
4. Configure parameters
5. Test in sandbox
6. Deploy

Each skill has:
- **Description** - What it does
- **Parameters** - Inputs required
- **Output** - What it returns
- **Rate Limits** - Usage restrictions
- **Cost** - Token fee per use
